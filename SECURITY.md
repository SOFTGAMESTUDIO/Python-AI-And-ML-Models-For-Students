# Security Policy

> **Provided by Soft Game Studio**

## Supported Versions

This repository, **Python AI & ML Models For Students**, is primarily an educational resource directed at students learning Artificial Intelligence and Machine Learning, rather than a production-grade enterprise software suite. However, we take code security seriously. We provide security patches and updates primarily for the `main` branch.

## Reporting a Vulnerability

If you discover a security vulnerability within any of our models, project algorithms, or dependencies, please do **NOT** report it through public GitHub issues.

Instead, please send a message to the Soft Game Studio team directly or open a private vulnerability report if supported by the platform. 

Please include:
* The specific project/file where the vulnerability exists.
* A detailed description of the vulnerability and its potential impact.
* Steps to reproduce the issue.

Our team will evaluate the problem and work towards a fix as quickly as possible to ensure our educational resources remain safe for students worldwide.
